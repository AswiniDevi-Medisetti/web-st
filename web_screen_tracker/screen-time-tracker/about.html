About ScreenTime

A lightweight front-end application that measures active time spent on the browser tab. It stores data locally to preserve privacy and allows export, pause, and visual analysis.

Project structure

  • Single-page styled pages with shared CSS & JS
  • LocalStorage data model per day (YYYY-MM-DD)
  • Page Visibility API + focus/blur detection for accurate sessions
  • Canvas-based chart for 7-day overview

Resources & references

MDN • Page Visibility API MDN • Canvas API MDN • localStorage

Upcoming features

  • Weekly insights — Smart suggestions based on your weekly active time patterns.
  • PWA support — Installable app + offline caching via Service Worker.
  • Profile & Sync — Optional end-to-end encrypted sync (opt-in).
  • Focus Mode — Pomodoro + auto-blocker integrations (experimental).
  • Better charts — Interactive charts & heatmaps for deep analysis.
These features are planned for future releases — keep an eye on the repo and demo videos below.
